# Obiettivo: Creare un'app web per prenotare online un trasporto sanitario per un familiare.

## Funzionalità

### Pannello Admin

#### Gestione utenti admin

* Creazione, modifica e cancellazione di utenti admin
* Recupero password
* Login con nome utente e password

#### Gestione clienti

* Visualizzazione, modifica e cancellazione di clienti
* Login con email e password

#### Gestione domande prenotazione

* Creazione, modifica e cancellazione di domandeeaaaaaa
* Domande a risposta singola o multipla, oppure a risposta aperta
* Attivazione/disattivazione di domande

#### Gestione prenotazioni trasporti

* Visualizzazione, modifica e cancellazione di prenotazioni
* Dettagli prenotazione:
    * Dati anagrafici paziente
    * Indirizzi di prelievo e destinazione
    * Risposte alle domande
    * Note
    * Stato (in validazione, validato, completato)

### Funzionalità Cliente

* Prenotazione di un nuovo trasporto
* Registrazione (in fase di prenotazione o con apposita schermata)
* Visualizzazione e modifica delle prenotazioni
* Prenotazione basata su una precedente (tranne le date)
* Annullamento prenotazione (con configurazione di giorni limite)
* Recupero password tramite email
* Richiesta di eliminazione account (con conferma da parte di un admin)

## Schermate

### Admin

* Login
* Recupero password
* Gestione prenotazioni
* Editor prenotazione
* Gestione domande
* Editor domanda
* Gestione clienti
* Editor clienti
* Gestione utenti admin
* Editor utenti

### Cliente

* Home (con scelta tra login e nuova prenotazione)
* Aggiunta nuova prenotazione (con email e password per utenti non registrati)
* Login
* Recupero password
* Lista prenotazioni
* Dettagli prenotazione (con possibilità di annullamento)
* Richiesta eliminazione account
